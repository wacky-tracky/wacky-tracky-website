📱 Mobile

Works on mobile - Android and iOS, phones, tablets, or anything with a modern web browser.

📁 Tags

Tag your tasks to make them easier to find later. You can tag things like "important", "friends" and "TODO!", make up any tag you like.

🖨 Printer friendly

Sometimes you just want to cross things off with a pen! Printer friendly format for everything.

🍺 Free as in "free beer"

The project is open source under GPLv2. We'll never charge you for the source code, so if you don't want a hosted version, you can run your own.

♿ Accessibility

The project takes accessibility really seriously. Very screenreader friendly.

📌 Unlimited levels of sub tasks!

You can have subtasks, then sub tasks of subtasks, then subtasks of subtasks of subtasks. All subtasks are first class tasks, and support tagging, notes, etc.

💁 You own your data

Get data in with import features, and get data out with export features. If you don't trust us, host your own copy of wacky-tracky. We're open and respect your freedom.

🕗 Due dates

When did you want to complete a task by? Set a Due Date using the easy date picker to find out.

📄 Rich Text Editor for notes

Sometimes you can't make everything into a task and you just have to write a wall of text. Because of this, every task has a big text block for notes too.

Developers & Techies

💻 REST API

Are you a developer? Well, wacky-tracky's server is designed from the ground up to act as a REST API. This means you could happily write third party clients/tools to use wacky-tracky.

💗 Open Source & Open Standards

Don't wait on companies to write code for you, don't get locked into their standards. Import and export your data from wacky-tracky.

This is the free, hosted version of wacky-tracky, but it is an open source project and techies can host their own version. If you're interested about the project go to wacky-tracky.org
